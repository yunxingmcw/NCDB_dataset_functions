# NCDB_dataset_functions
This include the extraction and preprocessing of the NCDB dataset.

The extraction is based on the 2022-PUF data structure.

Updated based on the NCDB Quick Data Structure PUF 2023 data structure; new length: 1095 characters, 6 new variables added: Tobacco Use Smoking Status (1048)
CLINICAL_MARGIN_WIDTH (1082-1085)
,HISTOLOGIC_SUBTYPE (1086)
RX_HOSP_SURG_PRIM_SITE_2023(1087-1090)
RX_SUMM_SURG_PRIM_SITE_2023(1091-1094)
and UR_CD_23 (1095)
